Phần mềm cần thiết
- Tableau desktop version 2023.3
- Python version 3 hoặc cao hơn
Cài đặt thư viện pandas python:
- pip install pandas
- pip install numpy

****HƯỚNG DẪN TĂNG CƯỜNG DỮ LIỆU****
B1: Giải nén folder chứa đồ án giữa kỳ
B2: Đặt chung file dataset gốc (dataset_full.csv) và file code python data_augmentation.py chạy cmd tại thư mục chứa hai file trên chạy file python, kết quả sẽ trả về 1 file dữ liệu mới tên là dataset_full_augmented.csv

****HƯỚNG DẪN CHẠY TABLEAU****
B1: Đặt file dataset_full.csv hoặc dataset_full_augmented.csv và file MarketsLatLong.xlsx chung với file Tableau workbook (midtermBI.twb)
B2: Mở file midtermBI.twb,  nếu Tableau báo lỗi yêu cầu đường dẫn thì ấn Edit Connection thay đỗi đường dẫn đến thư mục chứa các dataset ở bước 1. 
B3: Nhấn vào các worksheet ở dưới để xem từng view tương ứng với các câu hỏi ở bài giữa kì.
